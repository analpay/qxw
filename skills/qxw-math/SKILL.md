# qxw-math

字符串数学表达式计算器，安全（不用 `eval` / `exec`）。

## 基本用法

```bash
qxw-math "1 + 2 * 3"             # 务必加引号，避免 shell 解释 *、(、空格
qxw-math "2^10"                   # ^ 等价于 **
qxw-math "sqrt(2)"
qxw-math "√(9)"
qxw-math "√16"                    # √ 后直接跟数字也行
qxw-math "(3 + 4)**2 - 10"
echo "100 / 25" | qxw-math        # 从 stdin 读
```

## 参数

| 参数 | 缩写 | 默认 | 说明 |
|------|------|------|------|
| `<expression>` | - | 缺省时从 stdin 读取 | 表达式（可选位置参数） |
| `--quiet` | `-q` | false | 仅输出结果（纯数字），便于 `$(...)` 捕获 |

## 支持的语法

| 类别 | 运算符 / 语法 | 备注 |
|------|--------------|------|
| 加减 | `+` `-` | 一元 `+x` / `-x` 也支持 |
| 乘除 | `*` `/` | 真除，结果可能为浮点 |
| 整除 / 取余 | `//` `%` | 除数为 0 报错 |
| 次方 | `**` 或 `^` | `^` 会被归一化为 `**` |
| 开方 | `sqrt(x)` / `√(x)` / `√<数字>` | 负数开方报错；`√x`（裸变量）不支持 |
| 分组 | `( ... )` | 与 Python 同优先级 |

## 默认输出（Rich 表格）

```
$ qxw-math "(3+4)**2 - 10"
   数学表达式计算
┌────────┬──────────────┐
│ 表达式  │ (3+4)**2 - 10 │
│ 结果    │ 39            │
└────────┴──────────────┘
```

## 脚本场景

```bash
VAL=$(qxw-math -q "2^32")
echo "$VAL"                  # 4294967296

qxw-math -q "sqrt(2)"        # 1.4142135623730951（保留原始浮点精度）
```

## 退出码

| 退出码 | 触发场景 |
|--------|----------|
| 0 | 计算成功 |
| 2 | 未提供表达式且 stdin 是 TTY（没东西可算） |
| 6 | 语法错误 / 含不支持的节点 / 除零 / 负数开方 / 结果溢出（`ValidationError`） |
| 130 | 用户 Ctrl-C |
| 1 | 未预期的内部错误 |

## 安全性

底层是 `ast.parse` + 白名单遍历，**只放行**：

- 数字字面量
- `+ - * / // % **` 二元 / 一元运算
- 一组列在白名单里的函数（`sqrt`）

任何属性访问 / 变量名 / 列表 / 字典 / 字符串字面量 / 未授权的函数调用，都会在 AST 阶段被拒绝（退出码 6）。可以放心拿来处理用户输入。
