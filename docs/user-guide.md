# 使用手册

本文档介绍 QXW 工具集所有命令的详细用法。

## 命令概览

| 命令 | 说明 | 状态 |
|------|------|------|
| `qxw-hello` | 示例命令，验证安装 | ✅ 可用 |

## qxw-hello

示例命令，用于验证 QXW 工具集安装是否正确。

### 基本用法

```bash
# 命令行模式（默认）
qxw-hello

# TUI 交互模式
qxw-hello --tui
```

### 参数说明

| 参数 | 缩写 | 默认值 | 说明 |
|------|------|--------|------|
| `--name` | `-n` | 世界 | 问候对象的名称 |
| `--tui` | - | false | 启用 TUI 交互模式 |
| `--version` | - | - | 显示版本号 |
| `--help` | - | - | 显示帮助信息 |

### 示例

```bash
# 自定义问候对象
qxw-hello --name 开发者

# 查看版本
qxw-hello --version

# 查看帮助
qxw-hello --help
```

### TUI 界面快捷键

| 快捷键 | 功能 |
|--------|------|
| `Q` | 退出 |
| `D` | 切换暗色/亮色主题 |
